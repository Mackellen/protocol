我们收集什么？

简短回答：几乎不收集任何内容。 PassWallet 是一款完全本地化的应用，您的数据永远不会离开您的设备。

为了正常运行，我们仅处理：

  • ✓您主动输入的账户密码、银行卡信息、证件号码、安全笔记等数据（均存储在本地，仅您可见）
  • ✓您设置的主密码（使用 AES-256 本地加密存储，服务端无法解密）
  • ✓如果您开启生物识别解锁，系统会调用 Face ID / Touch ID 进行本地身份验证（我们不获取或存储生物特征数据）
  • ✓设备上生成的匿名崩溃日志（不含任何您输入的内容，仅用于提升应用稳定性）

我们不做什么？

以下是我们永远不会做的事：

  • ✗不收集、不上传任何密码或个人信息
  • ✗不使用任何第三方追踪器或分析 SDK
  • ✗不连接任何服务器或云服务（完全离线）
  • ✗不收集设备标识符、位置信息或通讯录
  • ✗不显示任何广告或推送通知
  • ✗不将您的数据用于任何形式的分析或训练

数据安全如何保障？

AES-256 加密

您的所有敏感数据均使用 AES-256 算法进行本地加密。这意味着即使设备落入他人手中，未经您的主密码解锁，数据仍然完全不可读。

主密码保护

主密码在设备本地经过 PBKDF2 密钥导出处理后，再用于加密您的数据。我们不会将主密码以任何形式传输、上传或存储。

生物识别安全

Face ID / Touch ID 仅用于在您设备本地验证身份。生物特征数据由系统 Keychain 保管，PassWallet 无法访问您的原始生物特征。

备份与数据导出

您的数据您做主

您可以随时将数据导出为加密备份文件。该备份文件同样经过加密处理，只有您当前的主密码才能解密。

我们建议您：

  • →定期导出加密备份，保存到安全的云存储（如 iCloud Drive 私有文件夹）
  • →备份文件与主密码不要保存在同一位置
  • →如果更换设备，使用备份恢复功能迁移数据
